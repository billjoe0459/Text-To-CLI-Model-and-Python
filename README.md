# Text-To-CLI-Model-and-Python
A Repository allowing for an AI model to generate and directly execute Linux Bash commands in an SSH terminal.\n
This project is powered by Python and Ollama \\add further notes here, python libs, etc...//\n
This project was initally prepaired and developed on a Jetson Orin Nano, future plans include extended training on an RTX 4070.\n\n
The based model used for this project is the Qwen/Qwen2.5-Coder-1.5B-Instruct pulled from huggingface.\n
The dataset used to initally train the model was aelhalili/bash-commands-dataset.\n
